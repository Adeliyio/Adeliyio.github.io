    End-to-End Stock Market Price Prediction Project

    This project is a web application that uses Streamlit to visualise stock data and forecast future stock prices. It retrieves historical stock data using the yfinance library, then applies Facebook's Prophet forecasting tool to predict future stock prices, which can be customized for a range of 1 to 4 years. The application presents interactive plots of both historical and forecasted data using Plotly, and also displays the components of the forecast such as trends and seasonality.
